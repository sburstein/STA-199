Does Lego set theme affect price? How well is piece count correlated with price of Lego sets across popular set themes?

piece_price_corr_viz <-
  lego_sales %>%
  filter(theme %in% c("Duplo", "Ninjago", "Star Wars", "Nexo Knights")) %>%
  
  ggplot(mapping = aes(x = pieces,
                       y = us_price,
                       color = theme)) +
    geom_point(na.rm = TRUE)
  
piece_price_corr_viz

Chose four of the most popular Lego set themes to analyze: “Duplo”, “Ninjago”, “Star Wars”, and “Nexo Knights”. Plotted the piece count on x-axis and set price on y-axis for sets across different Lego themes. There does appear to be a difference in cost based on set theme.

Another interesting takeaway is that the Duplo theme, while generally having small piece counts, is priced higher per piece than any of the other three set themes for many distinct piece counts.
